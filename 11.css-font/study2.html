font

  1. 사용할 폰트를 다운받는다
  2. 폰트를 등록한다
  3. 폰트를 태그에 적용한다

    4. @font-face { font-family: 사용할 폰트 이름 등록 (작업자가 작명); src: url("상대경로"); }
    적용할 태그 혹은 속성 { font-family: 작명한 이름과 동일하게 적용; }

    자의적으로 이름을 등록할 수 있다. 한글로 쓸 때는 ""를 사용해서 등록해준다, 영문일 때도 따옴표가 쓰이는데 그건 폰트명에 띄어쓰기가 들어가 있을 때 혼동을 방지하기 위해 ""가 쓰이는 것이다.
