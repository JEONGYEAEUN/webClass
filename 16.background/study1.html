background

태그의 공간에 배경색 또는 배경이미지를 나타낼 때 사용하는 css속성

배경은 컨텐츠가 아니고 단순 스타일이다 공간을 차지하지 않는다.

  • background-color(기본값:transparent) : 배경색상을 입력할 때 사용하는 속성
  • background_image(기본값:none) : 배경이미지를 입력할 때 사용하는 속성
  • background_repeat(기본값:repeat) : 입력된 배경이미지의 가로세로반복 속성을 제어하는 속성
  • background-position(기본값:0px 0px 0은 수치생략 가능) : 입력된 배경이미지의 위치를 x/y축값으로 제어하는 속성
  • background-size(기본값:auto auto) : 배경이미지의 가로세로사이즈를 지정하는 속성
  • background-blend-mode : 배경색 도는 배경이미지들의 혼합옵션을 지정하는 속성
  • background-attachment(기본값:scroll) : 배경이미지의 붙임값을 지정하는 속성

background-position

background-position속성은 기존 좌상단을 (0,0)좌표로 생각했을 때 좌표체계에 따라서 픽셀단위로 수치를 입력하여 배경이미지를 변경시키는 방법이 있다. 하지만 그런 방법만 통해서 배경이미지를 선택한 영역의 가로기준 중앙 세로기준 중앙에 배치하려고 하면 계산하는 과정이 복잡하기 때문에 키워드 방식으로도 배경이미지를 배치할 수 있다(left, right, top, bottom, center)

키워드 입력 방식으로만 배경이미지의 위치를 변경하기에는 표현에 제약이 많이 따르기 때문에 %단위를 사용해서도 배경이미지를 배치할 수 있다. 이 때, 좀 더 세부적인 위치변경을 하기위해서 cal()라는 css상의 계산기를 활용할 수 있다

cal() 괄호 안에는 +(더하기), -(빼기), X(곱하기), /(나누기)기호를 통해 계산을 할 수 있다. 이 때 더하기와 빼기 부호 앞뒤로 반드시 띄어쓰기를 해서 표현해야 하며, 곱셉 나눗셈은 띄어쓰기를 하지않아도 된다.

background-size

background-size:(가로사이즈) (세로사이즈) 속성을 통해서 배경이미지의 가로와 세로사이즈를 지정할 수 있다.

이 때, 가로사이즈를 픽셀이 아닌 %를 사용할 수 있다

그리고, auto라는 키워드를 통해서 원본이미지가 갖고있는 비율계산을 자동으로 적용해줄 수있다

  • cover : 원본 이미지의 가로세로 비율을 유지하면서 선택한 태그의 공간을 빈틈없이 가득 메꾸는 방식(이 때, 이미지가 잘려나가도 신경쓰지 않는 방식)
  • contain : 원본 이미지의 가로세로 비율을 유지하면서 선택한 태그의 공간을 최대한 가득 메꾸는 방식(이 때, 이미지가 절대 잘려나가지 않는 선에서 사이즈를 조절하는 방식)
