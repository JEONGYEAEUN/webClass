    • string
    • number
    • boolean
  • 참조데이터형태

자바스크립트언어를 통해서 홈페이지의 기능을 추가해줄 수 있다.

html은 정적인 언어이다. 그렇기 때문에 유동적인 컨텐츠를 표시해줄 수가 없다. 이를위해 자바스크립트 언어가 동적으로 작동하여 유동적인 컨텐츠를 보여준다.

변하는 컨텐츠를 보여주기 위해서 자바스크립트가 표현하는 6가지 데이터 형태를 공부해보자.

자바스크립트도 기본적으로는 위에서 아래 왼쪽에서 오른쪽으로 해석한다

자바스크립트에서 명령문 뒤에는 명시적으로 ;(세미콜론)을 써주어야 한다. 하지만 줄바꿈으로 하는 경우가 대부분이고 줄바꿈을 하면 자동으로 ;(세미콜론)이 붙은 것처럼 해석되기 때문에 생략이 가능하다

스크립트 태그는 자바스크립트에게 전달하는 명령을 넣는 공간

alert("내용") - 페이지에 알림창을 띄워주는 도구, 괄호 안에 창에 띄우고 싶은 문장을 작성한다

문자데이터 / string

문자데이터를 표현하기 위해서는 앞뒤로 쌍따옴표"" 혹은 홑따옴표''를 사용할 수 있다. 그리고 백틱``으로도 표현할 수 있다.

문자데이터 안에 들어가는 컨텐츠 내용으로 쌍따옴표 혹은 홑따옴표가 사용될 수 있기 때문에 문자데이터의 표식은 3가지 형태가 존재한다.

문자데이터는 +(덧셈연산문자)를 통해서 서로 이어줄 수 있다

숫자데이터 / number

숫자데이터는 사칙연산이 가능하다

숫자데이터는 따옴표를 사용하지 않는다

수학연산은 덧셈 뺄셈보다 곱셈 나눗셈이 우선계산된다. 덧셈 뺄셈을 우선계산하고싶다면 명시적으로 ()괄호를 사용하여 묶어준다

0으로 시작하는 숫자는 존재하지 않는다

문자데이터와 숫자데이터를 더하면 서로 이어지면서 결과적으로는 문자데이터가 된다.

  • +(덧셈)
  • -(뺄셈)
  • *(곱셈)
  • /(나눗셈)
  • %(나머지연산)
