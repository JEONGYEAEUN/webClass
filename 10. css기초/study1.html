CSS

html 태그를 스타일링하는 언어- html파일이 전제가 되어야한다

방식

  • inline-style(element-style)

    태그를 직접 선택하여 시작태그에 속성을 부여하여 스타일을 입력하는 방식

  • 내부스타일시트

    head태그 안에 style태그를 작성하여 스타일을 입력하는 방법

  • 외부스타일시트

    html파일 바깥에 별도의 css파일을 생성해서 스타일을 입력하는 방식

css기초문법

css는 태그를 선택해서 스타일을 입력하는 언어 그렇기 때문에 선택하는 부분과 스타일이 입력되는 부분으로 나뉜다.

선택(문)자{스타일속성:스타일속성값;스타일속성:스타일속성값;}

선택자(태그명,아이디명,클래스명으로 선택할 수 있다)

style태그 안에는 순순한 css문법만 사용될 수 있다.

여러가지 스타일 정보를 입력할 때 ; 로 구분해준다 - 구분자로 사용된다.
